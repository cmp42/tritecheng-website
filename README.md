Tritech Engineering Corporation Website

Overview
This repository contains the official website for Tritech Engineering Corporation, a civil engineering and surveying firm based in Dover, New Hampshire.
The site is a static HTML/CSS website designed to be lightweight, fast, and easy to maintain.

The website includes:
Company overview and branding
Services (Civil Engineering, Surveying, Land Planning)
Projects (Residential, Commercial, Municipal)
Careers information
Contact information with location map

The site is currently deployed via a static hosting service for preview and testing. 
The website is currently accessible at: 
http://tritechengcorp.netlify.app 

Once DNS and hosting are finalized, the website will be published under Tritech's official domain: 
http://www.tritecheng.com
